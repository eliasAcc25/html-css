Elemento Text-media

Flexbox é uma abordagem de layout flexível e responsiva que permite arranjar itens dentro de um contêiner de forma flexível. Ele é ideal para layouts de uma dimensão, como linhas ou colunas, e é utilizado para criar layouts flexíveis e responsivos sem a necessidade de usar floats ou posicionamento. Flexbox é direcionado, o que significa que ele pode alterar a largura e a altura dos itens para preencher o espaço livre disponível, adaptando-se a diferentes dispositivos e tamanhos de tela.
